# Mundo Alfabeto

Aplicacion web educativa para aprender letras y palabras de forma interactiva.

## Como correr el proyecto

1. Instalar dependencias:
   `npm install`
2. Ejecutar en desarrollo:
   `npm run dev`
3. Abrir:
   `http://localhost:3000`

## Ideas de features futuras

- Progreso por usuario con niveles y estadisticas.
- Juegos nuevos (memoria, completar palabras, arrastrar y soltar).
- Audio de pronunciacion por letra y palabra.
- Modo docente/padres con reportes de avance.
- Soporte multi idioma (espanol, ingles, portugues).
- Version mobile con PWA para uso offline.
